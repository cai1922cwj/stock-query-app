# 股票查询APP

一款支持A股、港股、美股实时行情查询的移动端Web应用。

## 功能特性

- 📈 **实时行情** - 支持A股、港股、美股实时行情查询
- 🔍 **股票搜索** - 快速搜索股票代码或名称
- ⭐ **自选股** - 添加关注的股票到自选列表
- 📊 **市场概览** - 查看主要市场指数和涨跌榜
- 📱 **移动端适配** - 完美适配手机浏览器
- 🌐 **PWA支持** - 可安装为桌面应用

## 技术栈

- Vue 3 + Composition API
- Vue Router
- Pinia
- Vite
- ECharts
- Axios

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 股票代码格式

- A股: `sh600519` (上海) / `sz000858` (深圳)
- 港股: `hk00700`
- 美股: `usAAPL`

## 部署

构建后的文件位于 `dist` 目录，可部署到任何静态服务器。
